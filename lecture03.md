# AP サーバーについて調べてみましょう。
1. AP サーバーの名前とバージョンを確認してみましょう。
   APサーバ名：puma Version 5.6.5 →　APサーバ.png
2. AP サーバーを終了させた場合、引き続きアクセスできますか？
    接続できず　→　APサーバ終了時.png
4. 結果を確認して、また AP サーバーを起動してください。
    確認後再接続　→　APサーバ再起動.png
